CSS3 Animations

Overview

Gradually change an elements style from one to another with keyframe animations.

Demo

Code

  • html
  • CSS

<div class="example"></div>

@keyframes example {
    0% {background-color: #efefef; margin-left: 0px;}
    50% {background-color: #374252; margin-left: calc(100% - 100px);}
    100% {background-color: #efefef; margin-left: 0px;}
}

.example {
background-color: #efefef;
border-radius: 100%;
height: 100px;
animation-name: example;
animation-duration: 8s;
animation-iteration-count: infinite;
animation-timing-function: ease-in-out;
width: 100px;
}
